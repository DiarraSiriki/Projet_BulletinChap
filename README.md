##Présentation


  BulletinChap  est une plateforme web qui permet aux élèves de consulter leur bulletin scolaire 
  en ligne, où qu’ils soient, sans avoir à se déplacer dans leur établissement.
  Ce projet s’inscrit dans une démarche de digitalisation du système 
  éducatif, en facilitant l’accès aux résultats scolaires de manière simple, rapide et sécurisée.

##objectifs

  Permettre aux élèves d’accéder à leurs bulletins à distance
  Réduire les déplacements et les files d’attente dans les établissements
  Moderniser la gestion des résultats scolaires
  Proposer une solution numérique adaptée au contexte éducatif

##Fonctionnalités

  -saisie du matricule de l’élève
  -afichage du bulletin scolaire
  -calcul automatique des moyennes
  -Message d’erreur en cas de matricule incorrect

##Fonctionnement

  L’utilisateur entre son matricule dans un formulaire.
  Le système recherche les informations correspondantes
  dans une base de données simulée (JavaScript).

  Si l’élève est trouvé :
  Il est redirigé vers la page bulletin
  Ses informations et ses notes sont affichées

 Sinon :
 Un message d’erreur est affiché
